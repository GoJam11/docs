## 系统要求

### Windows
- Windows 10 或更高版本
- 64位系统

### macOS
- macOS 10.15 或更高版本
- 支持 Intel 和 Apple Silicon 芯片

### Linux
- Ubuntu 18.04 或更高版本
- 其他主流 Linux 发行版

## 安装说明

### Windows
1. 下载 `.exe` 安装文件
2. 双击运行安装程序
3. 按照安装向导完成安装

### macOS
1. 下载 `.dmg` 文件
2. 双击打开 DMG 镜像
3. 将 Fluent Desk AI 拖拽到应用程序文件夹

### Linux
根据您的发行版选择对应的安装方式，或使用通用的 tar.gz 包。

## 开始使用

安装完成后，您可以：

1. 启动 Fluent Desk AI 应用程序
2. 参考我们的[快速开始指南](/quickstart)进行配置
3. 查看[使用指南](/guide/setup)了解更多功能
