### 快捷键支持（计划中）
- `Ctrl/Cmd + N`: 新建对话
- `Ctrl/Cmd + T`: 切换模型

- `Ctrl/Cmd + Shift + P`: 快捷 Prompt 面板
- `Ctrl/Cmd + Enter`: 发送消息

### 模型参数调节（计划中）
- **Temperature**: 控制回答的创造性（0-2）
- **Max Tokens**: 限制回答的最大长度
- **Top-p**: 控制回答的多样性（0-1）
- **Frequency Penalty**: 减少重复内容
- **Presence Penalty**: 鼓励主题多样性
