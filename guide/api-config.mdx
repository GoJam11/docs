FluentAI 支持多种主流 AI 服务商，本指南将详细介绍如何配置各种 API 密钥。

## 配置入口

1. 打开 FluentAI 应用
2. 进入 **首页 → 偏好设置 → API设置**
3. 点击 **"添加供应商"**

<img
  src="/images/api-settings-entry.png"
  alt="API设置入口"
  
/>

## 支持的服务商类型

FluentAI 支持三种类型的服务商配置：

### 自定义供应商
适用于有自定义 API 端点的服务商

<Steps>
  <Step title="填写基本信息">
    输入供应商名称和 API 基础 URL
  </Step>
  <Step title="添加 API 密钥">
    输入您的 API 密钥
  </Step>
  <Step title="获取模型列表">
    点击"获取全部模型"按钮
  </Step>
  <Step title="添加自定义模型">
    选择"添加自定义模型"完成配置
  </Step>
</Steps>

  src="/images/custom-provider-config.png"

## 重要提示

<Warning>
**API 密钥安全**
- 请勿在公共场所输入 API 密钥
- 不要与他人分享您的 API 密钥
- 定期检查 API 使用情况和账单
</Warning>

<Tip>
**配置建议**
- 建议优先配置 DeepSeek，性价比较高
- OpenAI 模型功能丰富但费用较高
- Ollama 适合对隐私要求较高的用户
</Tip>

## 常见问题

### Q: API 密钥在哪里获取？
A: 每个服务商都有自己的开发者平台：
- **OpenAI**: https://platform.openai.com/api-keys
- **DeepSeek**: https://platform.deepseek.com/api-keys
- **Anthropic**: https://console.anthropic.com/

### Q: 配置后无法使用怎么办？
A: 请检查：
1. API 密钥是否正确
2. 账户是否有足够余额
3. API 密钥是否有相应权限
4. 网络连接是否正常

### Q: 可以同时配置多个供应商吗？
A: 是的！FluentAI 支持同时配置多个供应商，您可以在不同场景下切换使用不同的模型。
