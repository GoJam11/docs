# 模型选择与管理

配置好 API 后，下一步就是选择合适的 AI 模型。FluentAI 支持多种主流模型，您可以根据需求灵活选择。

## 🎯 添加模型

### 配置完成后选择模型

1. **完成 API 配置**：确保您已经配置了至少一个 AI 服务商
2. **查看可用模型**：系统会自动显示该服务商的所有可用模型
3. **选择需要的模型**：从列表中选择您想要使用的模型

<Steps>

## ⚙️ 模型管理

### 切换模型
- 在对话界面可以随时切换不同的模型
- 不同模型有不同的特点和定价
- 建议根据任务类型选择合适的模型

### 模型参数调整
某些模型支持参数调整：
- **Temperature**: 控制回答的创造性
- **Max Tokens**: 限制回答长度
- **Top-p**: 控制回答的多样性

## 🔄 模型切换技巧

<Tip>
**智能选择策略**
- 简单问答：使用 GPT-3.5 或 DeepSeek
- 复杂分析：使用 GPT-4 或 Claude 3 Opus
- 代码相关：优先选择 DeepSeek Coder
- 图像理解：使用支持视觉的多模态模型
</Tip>

## ⚠️ 注意事项

<Warning>
**成本控制**
- 不同模型的计费标准不同
- GPT-4 系列相对昂贵，建议按需使用
- 定期检查 API 使用量和费用
</Warning>

<Note>
**性能优化**
- 选择离您最近的服务器节点
- 避免在高峰期使用高耗时模型
- 合理设置 Token 限制
</Note>
