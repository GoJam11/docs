## 下载和安装

1. 点击页面右上角的 **"立即下载"** 按钮
2. 下载适合您系统的安装包
3. 按照系统提示完成安装
4. 启动 FluentAI 应用程序

## 快速配置

### 推荐配置方案

<Tabs>
  <Tab title="新手推荐">
    **DeepSeek（推荐）**
    - 性价比最高
    - 中文支持优秀
    - 配置简单

    <Steps>
      <Step title="获取 API 密钥">
        访问 [DeepSeek 平台](https://platform.deepseek.com) 注册并获取 API 密钥
      </Step>
      <Step title="在 FluentAI 中配置">
        打开设置 → API设置 → 添加供应商 → 选择 DeepSeek
      </Step>
      <Step title="输入密钥">
        粘贴您的 API 密钥并保存
      </Step>
    </Steps>
  </Tab>

  <Tab title="专业用户">
    **多模型配置**
    - OpenAI GPT-4：复杂分析任务
    - Claude 3.5：长文档处理
    - DeepSeek：日常对话

    建议同时配置 2-3 个服务商，根据任务类型灵活切换。
  </Tab>

  <Tab title="隐私优先">
    **Ollama 本地部署**
    - 完全本地运行
    - 数据不出本地
    - 需要较高硬件配置

    <Steps>
      <Step title="安装 Ollama">
        访问 [Ollama 官网](https://ollama.ai) 下载安装
      </Step>
      <Step title="下载模型">
        运行 `ollama pull llama2` 或其他模型
      </Step>
      <Step title="连接 FluentAI">
        在 FluentAI 中选择 Ollama 供应商
      </Step>
    </Steps>
  </Tab>
</Tabs>

## 故障排除

### 常见问题快速解决

<AccordionGroup>
  <Accordion icon="key" title="API 密钥无效">
    1. 检查密钥是否复制完整
    2. 确认密钥是否已激活
    3. 验证账户余额是否充足
    4. 检查 API 权限设置
  </Accordion>

  <Accordion icon="wifi" title="网络连接问题">
    1. 检查网络连接
    2. 尝试切换网络环境
    3. 检查防火墙设置
    4. 确认代理配置正确
  </Accordion>

  <Accordion icon="loader" title="模型响应慢">
    1. 尝试切换到更快的模型
    2. 减少单次请求的内容长度
    3. 检查 API 服务状态
    4. 优化网络环境
  </Accordion>
</AccordionGroup>
