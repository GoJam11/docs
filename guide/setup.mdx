# 快速开始

欢迎使用 FluentAI！本指南将帮助您快速完成安装和初始配置。

## 📥 下载安装

### Windows 用户
1. 点击页面顶部的 "下载 Windows 版" 按钮
2. 下载完成后运行安装程序
3. 按照安装向导完成安装

### Mac 用户
1. 点击页面顶部的 "下载 Mac 版" 按钮
2. 下载完成后打开 .dmg 文件
3. 将 FluentAI 拖拽到应用程序文件夹

## 🚀 首次启动

启动 FluentAI 后，您需要完成以下步骤：

<Steps>
  <Step title="配置 API">
    首次使用需要配置 AI 服务商的 API 密钥
  </Step>
  <Step title="选择模型">
    从支持的模型中选择您需要的 AI 助手
  </Step>
  <Step title="开始对话">
    一切准备就绪，开始您的 AI 对话之旅
  </Step>
</Steps>

## 💡 小贴士

<Tip>
首次使用建议先配置一个您熟悉的 AI 服务商（如 OpenAI 或 DeepSeek），这样可以快速体验 FluentAI 的功能。
</Tip>

<Warning>
请妥善保管您的 API 密钥，不要在公共场所或与他人分享。FluentAI 会安全地存储您的密钥信息。
</Warning>
